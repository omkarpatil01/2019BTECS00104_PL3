Code to find Armstrong number in the given range

Armstrong Number: It is a number that is equal to the sum of the cubes of its digits.
For example 153.
Sum of the cubes of its digit = 13 + 53 + 33 = 1 + 125 + 27 = 153

Sample code

            
                #include 
                #include 
                using namespace std;

                int main()
                {
                    int r, sum, temp, ans;
                    int x1, x2;
                    cout << "\n\n Find the Armstrong number for a given range of number:\n";
                    cout << " Input starting number of range: ";
                    cin >> x1;
                    cout << " Input ending number of range: ";
                    cin >> x2;
                    cout << " Armstrong numbers in given range are: " << endl;
                    for (int i = x1; i <= x2; i++) 
                    {
                        temp = i;
                        sum = 0;
                        while (temp != 0) 
                        {
                            r = temp % 10;
                            ans = pow(r, 3);
                            sum = sum + ans;
                            temp = temp / 10;
                        }
                        if (sum == i)
                            cout << i << " ";
                    }
                    cout << endl;
                }        
            
            
                
Sample Input:

                    Find the Armstrong number for a given range of number:                            
                    Input starting number of range: 25                                    
                    Input ending number of range: 200 
                
Sample output

                    Armstrong numbers in given range are:                                 
                    153

Variables used in program-

In the above code total 7 variables used.

x1 stores the starting number of range.
x2 stores end number of range.
i is used for iteration over the range.
temp is tempoarary variable stores current number.
sumis used to calculate sum of number.
r stores remainder by modulo operation.
ans stores cube of digit.

Time Complexity-

Since there 1 nested loop so time complexity of above code is O(n 2 )
